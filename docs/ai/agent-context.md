# AI Agent Context

## 项目定位

MeshLink 是一个 WireGuard SD-WAN 项目，当前阶段目标是尽快做出可运行的 MVP，而不是一次性做全量平台。

## 默认优先级

1. 协议清晰
2. 模块边界清晰
3. Linux 先行
4. 可验证优先于全面覆盖
5. 文档与代码同步

## AI Agent 必须知道的约束

1. 服务端不转发正常业务流量。
2. 客户端自己做路径选择和打洞。
3. Relay 是兜底路径，不是默认路径。
4. 协议和文档是后续自动生成代码的重要输入，不允许随意漂移。

## AI Agent 修改代码前必须检查

1. 是否影响 `proto/`
2. 是否影响 `docs/api/`
3. 是否影响 `docs/architecture/`
4. 是否需要补充 `docs/ai/` 中的任务边界
